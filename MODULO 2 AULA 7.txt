===================== WEB SERVER ===============================

Existem 2 tipos: estático e dinâmico. O estático é um servidor físico onde são armazenados arquivos, softwares e/ou banco de dados. O dinâmico se refere aos softwares que estão presentes no servidor físico.

* Os tipos de Web Server dinâmicos são:
	Arquivos (file server) [armazena arquivos]
	Aplicações (application server) [gerencia dados e aplicações]
	Banco de dados (database)
	All (todos juntos)

* Os dados precisam estar localizados em um servidor.
